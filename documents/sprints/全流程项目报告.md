## 项目背景与阶段性里程碑

《杯级计算器》从一个轻量级的 Python 脚本，经过多个迭代周期（Sprints），逐步演化为了包含自动抓取构建、复杂伤害面向对象体系、FastAPI 本地服务、桌面 Web 视图的闭环可视化应用程序。

## 各阶段冲刺摘要

### ✅ Sprint 1: 后端开发与计算核心搭建

- **进度**：100% 完成
- **核心产出**：
  - 构建了四大属性修饰器与五大伤害类型的基座逻辑。
  - 完成大模型 Prompt 流水线，实现了 `char_xxx.json` 到独立 `operator_xxx.py` 的 AST 动态代码生成。
  - 实现基于 FastAPI 的基础数据下发接口。
- **重点突破**：干员多变机制与属性计算的统一抽象；`importlib` 实现全量代码的动态映射与加载。

### ✅ Sprint 2: 前端开发与视觉重构

- **进度**：100% 完成
- **核心产出**：
  - 完成了具备高级暗黑风格与毛玻璃交互质感的可视化 Web 页面。
  - 实现了干员大职业与分支下拉级联筛选菜单。
  - 交付了 Sandbox（沙盒模拟）、全境演算（团队排名）等功能视图布局。
- **重点突破**：通过原生 Vanilla CSS/JS 打造无需构建流的前端方案；规避原生 JS 导致的 DOM 大量重绘卡顿。

### ✅ Sprint 3: 前后端联通及数据互通

- **进度**：100% 完成
- **核心产出**：
  - 引入了 `pywebview` 壳，实现了 Python 服务端在后台启用的同时唤起客户端窗口。
  - 前端 JS 与 FastAPI 后端通过 `127.0.0.1` 完成了跨域端口与请求的数据互联。
  - 沙盒演算数据结果成功被解析并可视化。
- **重点突破**：解决端口被占用的容错问题，确保后台与前台渲染生命周期的同步。

### ✅ Sprint 4: 细节优化与封装发布

- **进度**：100% 完成
- **核心产出**：
  - 代码异常防护：对 `w`、`麒麟R夜刀`、`林` 等极端机制引发的 `list index out of range` 和大小写匹配异常进行了健壮性修复。
  - 执行工程化打包：引入了 `path_resolver.py`，配置了 `build_app.py` 自动化打包管线。
  - 完成了 `.exe` 独立应用程序的封装交付。
- **重点突破**：彻底解决 PyInstaller `_MEIPASS` 特殊环境下的多静态资源和动态 Python 类反射加载失效的难题。
